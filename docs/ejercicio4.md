# 4. Recuperación de audios en memoria

Se implementó un Servicio Genérico (storage.ts) para manejar la lectura y escritura de objetos JSON en el disco.

Al cargar la aplicación, un useEffect recupera las URIs de sesiones anteriores, permitiendo que el usuario vea sus audios previos inmediatamente
